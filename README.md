# Myntra_Frontend_React_Redux
Developed a responsive single-page frontend application inspired by Myntra, focusing on core e-commerce user flows. The application displays a product listing page with dynamic item rendering, supports adding/removing items from the cart, and shows a dedicated bag page with real-time price calculation including discounts. Implemented global state management using Redux and integrated a loading spinner to enhance user experience during data fetching.

Tech Stack:
React.js
Redux (Redux Toolkit)
JavaScript (ES6)
HTML5
CSS3

Key Features:
Product listing page with dynamically rendered items
Add to Cart and Remove from Cart functionality
Bag/Cart page displaying selected items
Real-time total amount calculation with discount applied
Global state management using Redux store
Loading spinner displayed while fetching product data
Single Page Application (SPA) with smooth UI updates
Responsive layout for better user experience
